# Анализ воронки авиабилетов и дизайн A/B-теста  
**Даниил Рыбаков**

## О проекте
Разведочный анализ датасета «Воронка авиабилетов» и разработка A/B-теста для роста конверсии. Проект был реализован в рамках прохождения интенсива по Продуктовой Аналитике от Т-Банка.

В проекте:
- EDA(Разведочный анализ данных) в Jupyter Notebook  
- Анализ узких мест воронки  
- Формулирование продуктовых гипотез  
- Дизайн A/B-теста  

---

## Структура проекта

- `Avia_convers.ipynb` - EDA  
- `Avia_convers.pdf` - краткая аналитическая записка
- `dano_dataset_avia_hashed_corrected_s.xlsx` - Датасет действий пользователей в сервисе покупки Авиабилетов
